# Code of Conduct

This project adopts the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
unchanged and in full.

The canonical text lives at the URL above; it is not copied here to
avoid drift between our copy and the upstream revision.

## Enforcement contact

Please report abuse or conduct issues to the project maintainer at
`hctsai1006@cs.nctu.edu.tw`. Reports will be reviewed in confidence and
responded to within seven days.

## Scope

The Code of Conduct applies to every space managed by this project:
the GitHub repository, the issue tracker, the discussions tab (if
enabled), pull request reviews, release notes, and any off-platform
interactions that concern the project (conference talks, mailing-list
threads, etc.).

## Enforcement

Actions taken by maintainers in response to a confirmed violation may
include (non-exhaustive):

1. A private written warning.
2. A public warning in the relevant thread.
3. Temporary removal from project spaces (e.g. a 30-day ban).
4. Permanent removal for repeated or severe violations.

Decisions are at the maintainer's discretion and are final for the
governance of this repository. Escalation to the NYCU WinLab
sponsoring faculty is available if a dispute cannot be resolved.
